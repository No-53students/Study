# 数组与字符串

数组和字符串是最基础的数据结构，也是面试中最常见的题型。掌握各种数组操作技巧对于解决算法问题至关重要。

## 核心技巧

### 1. 双指针法

双指针是数组问题中最常用的技巧，主要有以下几种形式：

- **快慢指针**：一个指针遍历，另一个指针记录有效位置
- **左右指针**：从两端向中间逼近
- **滑动窗口**：维护一个满足条件的窗口

```javascript
// 快慢指针示例：移除元素
function removeElement(nums, val) {
  let slow = 0;
  for (let fast = 0; fast < nums.length; fast++) {
    if (nums[fast] !== val) {
      nums[slow++] = nums[fast];
    }
  }
  return slow;
}
```

### 2. 原地修改

很多数组题目要求 **原地修改**，即空间复杂度 O(1)。关键是利用数组本身的空间，通过交换或覆盖来完成操作。

### 3. 贪心算法

贪心算法在每一步都做出当前最优的选择，常用于：

- 买卖股票问题
- 跳跃游戏
- 区间调度

### 4. 前缀和/后缀积

预处理数组，快速计算区间和或区间积：

```javascript
// 前缀和
const prefix = [0];
for (let i = 0; i < nums.length; i++) {
  prefix.push(prefix[i] + nums[i]);
}
// 区间 [i, j] 的和 = prefix[j+1] - prefix[i]
```

## 常见题型

| 题型 | 难度 | 核心技巧 |
|------|------|----------|
| 移除元素 | 简单 | 快慢双指针 |
| 删除重复项 | 简单 | 快慢双指针 |
| 合并有序数组 | 简单 | 逆向双指针 |
| 多数元素 | 简单 | 摩尔投票 |
| 轮转数组 | 中等 | 三次翻转 |
| 买卖股票 | 简单/中等 | 贪心/动态规划 |
| 跳跃游戏 | 中等 | 贪心 |
| 接雨水 | 困难 | 双指针/单调栈 |

## 时间复杂度对比

| 操作 | 数组 | 字符串 |
|------|------|--------|
| 访问 | O(1) | O(1) |
| 搜索 | O(n) | O(n) |
| 插入 | O(n) | O(n) |
| 删除 | O(n) | O(n) |

## 注意事项

1. **边界条件**：空数组、单元素数组
2. **整数溢出**：大数相加/相乘
3. **字符串不可变**：JavaScript 中字符串不可变，需要转为数组操作
4. **原地修改**：注意是否允许使用额外空间
